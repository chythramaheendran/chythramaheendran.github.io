PHASE 01
Data Extraction
750 rows across 12 Excel sheets reconciled to the rupee — fixing date serials, null categories, and refund entries.
PHASE 02
Local Dashboard
Single HTML file with charts, settlement calculator, collapsible transactions, and 5 trip breakdowns. No install required.
PHASE 03
Firebase Integration
Firestore database with onSnapshot listeners for instant sync. First-run seeding uploads all 750 records in Firestore batches.
PHASE 04
Access Control
Owner / Editor / Viewer roles by Gmail address. Anyone not added is blocked at sign-in. Roles sync to Firestore in real time.
PHASE 05
Mobile Responsiveness
Fixed bottom sheet overflow, tab switching bug (display:flex not ""), and Google sign-in across Android and iOS.
PHASE 06–07
Desktop + Config
220px sidebar layout, modal sheet on desktop. Inline ✏️ editor for categories and payment modes inside the Add Expense form.
